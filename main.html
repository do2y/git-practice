Fresh Layout

보기 좋은 구성이 연습도 더 재밌게 만듭니다.

제목, 설명, 버튼, 카드, 포인트 색상을 한 화면에 균형 있게 배치했습니다. 포트폴리오 느낌으로 확장하기도 쉬운 구성입니다.

Focus Warm + Clean

부드러운 배경 위에 또렷한 포인트 컬러를 올려서 시선이 잘 모이게 했습니다.

Highlights

이 페이지에서 연습할 수 있는 것

레이아웃, 카드 UI, 버튼 스타일, 타이포 크기 조절, 배경 그라데이션을 한 번에 만져볼 수 있습니다.

  • HTML 구조 연습 header, section, article, aside 같은 시맨틱 태그 사용
  • CSS 감각 익히기 색 조합, 그림자, radius, spacing 조절
  • 반응형 확인 창 크기를 줄였을 때 레이아웃이 어떻게 바뀌는지 테스트
